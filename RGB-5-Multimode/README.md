# 🌈 RGB5 Multi Mode LED Project

A simple Arduino project that controls **5 RGB LEDs** with multiple lighting modes and smooth transitions.

---

## 📌 Description
This project uses an Arduino to control **five RGB LEDs**.  
Each LED can display different colors like **Red, Green, Blue**, and combinations of these colors.  
The LEDs run in **multiple modes** such as color switching, patterns, and smooth fade in/out effects.

---

## ✨ Features
- 🔴 Multiple color modes  
- 🌈 RGB color combinations  
- 🔁 Different lighting patterns  
- ✨ Smooth fade in & fade out effect  

---

## 🧰 Components Used
- Arduino Board  
- 5 × RGB LEDs  
- Resistors  
- Jumper Wires

## ⚙️ How It Works
- Arduino controls all 5 RGB LEDs  
- Different modes are programmed for various lighting effects  
- Smooth transitions are created using timing (software PWM)

---

## 🚀 How to Use
1. Upload the code to Arduino  
2. Connect the circuit properly  
3. Power the system  
4. Enjoy different lighting modes
