# Multiple-LED-Blink
Control multiple LEDs using Arduino Nano

## This project demonstrates how to blink multiple Leds using Ardiuno.
  It is a beginner-friendly electronice project.
  
## Components
- Arduino UNO/NANO
- 5mm LEDS (10 Pieces)
- 220 Ohm Resistors
- Breadboard
- jumper Wires

## Circuit Diagram
Connect LEDs to digital pins 13 to 4 no pin and 220 Ohm resistors.
GND goes to Arduino GND.

## How to Run
1. Open Arduino IDE
2. Copy the code from 
3. Connect Arduino to PC via USB
4. Select the correct board and port
5. Click Upload

## Code Explanation

This Arduino code controls 10 LEDs with multiple blinking patterns:

1. **All LEDs On/Off together** – First, all 10 LEDs turn on at the same time, then turn off.
2. **Sequential Blink** – LEDs light up one by one in sequence, then turn off one by one.
3. **Rapid Blink** – All LEDs blink rapidly together, creating a flashing effect.
4. **Multiple Fun Modes** – The code cycles through various blinking patterns for a dynamic LED display.

Each pattern uses delays to control the speed of blinking.
